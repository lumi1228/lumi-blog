## Koa简介
### 什么是Koa?
Koa 是一个新的 web 框架，致力于成为 web 应用和 API 开发领域中的一个更小、更富有表现力、更健壮的基石。  
利用 async 函数丢弃回调函数，并增强错误处理。 Koa 没有任何预置的中间件，可快速而愉快地编写服务端应用程序。

### 核心概念
- Context（上下文）
- Koa Application（应用程序）
- Request（请求）、Response（响应）
